# Reddit 使用指南

## 如何发布帖子

### 步骤 1: 选择社区
点击右上角的 "+ Create Post" 按钮，选择一个合适的社区（subreddit）。

### 步骤 2: 编写内容
在编辑框中输入你的帖子标题和内容。确保内容符合社区规则。

### 步骤 3: 添加标签
为你的帖子添加相关的标签（flair），帮助其他用户找到你的内容。

### 步骤 4: 发布
点击 "Post" 按钮发布你的帖子。

## 如何积累粉丝

### 策略 1: 选择垂直社区
选择 3-5 个你感兴趣的垂直社区，专注在这些社区中发布高质量内容。

### 策略 2: 每天发布原创内容
保持每天发布 1-2 篇高质量的原创内容，确保内容有价值。

### 策略 3: 深度互动
在热门帖子下发表有深度的评论，展示你的专业知识和见解。

### 策略 4: 建立真实互动
回复其他用户的评论，参与社区讨论，建立真实的社交关系。

### 策略 5: 分析数据优化
使用 Reddit 的分析工具，了解最佳发布时间和内容类型。

### 策略 6: 持续建立品牌
坚持 3-6 个月，持续发布有价值的内容，建立个人品牌。

## UI 元素说明

- "Create Post" 按钮：位于页面右上角，用于创建新帖子
- 搜索框：位于顶部导航栏，可以搜索 Reddit 内容
- 社区信息卡片：位于右侧边栏，显示社区统计信息
- 评论按钮：每个帖子下方，用于查看和发表评论
